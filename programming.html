Προγραμματισμός με Java

Μάθετε περισσότερα για το αντικείμενο

Δομή ενός προγράμματος Java και εντολές ελέγχου. Κλάσεις και αντικείμενα: συναρτήσεις δημιουργίας, μεταβλητές στιγμιότυπου, στατικές μεταβλητές, αναφορές σε αντικείμενα.

- 2 ώρες / βδομάδα

- 50€ / 6 μήνες

- Περιεχόμενο μαθήματος:

  • Εισαγωγή στον αντικειμενοστραφή προγραμματισμό
  • Μέθοδοι: εμβέλεια μεθόδων, απόκρυψη κώδικα, υπερφόρτωση μεθόδων, στατικές μέθοδοι
  • Δομές δεδομένων και συλλογές: αυτό-αναφερόμενες κλάσεις και δυναμική δέσμευση μνήμης, συνδεδεμένες λίστες, πίνακες και τα λοιπά
  • Σχεδίαση με αντικειμενοστραφή προγραμματισμό: προσδιορισμός και υλοποίηση σχέσεων μεταξύ κλάσεων
  • Κληρονομικότητα και πολυμορφισμός: βελτιστοποίηση ιεραρχίας κλάσεων και επαναχρησιμοποίηση κώδικα, πολυμορφική επεξεργασία και δυναμική δέσμευση μεθόδων, αφηρημένες και τελικές κλάσεις και μέθοδοι
  • Γενικεύσεις και Χειρισμός εξαιρέσεων
  • Επανάληψη όσων έχουμε μάθει με παραδείγματα με την χρήση προγραμμάτων που υπάρχουν και χρησιμοποιούνται

- 2 ώρες / βδομάδα

- 50€ / 6 μήνες

- Περιεχόμενο μαθήματος:

  • Εισαγωγή στο αντικείμενο
  • Δυαδικά δέντρα αναζήτησης: ορισμός, αναζήτηση, εισαγωγή, διαγραφή
  • Ισοζυγισμένα δέντρα αναζήτησης: δέντρα AVL, δέντρα 2-3, ερυθρόμαυρα δέντρα, δέντρα Β

  • Συνδεδεμένες λίστες: μονή, διπλή και κυκλική σύνδεση, διάσχιση, εισαγωγή, διαγραφή.
  • Δυαδικοί σωροί: ορισμός, λειτουργίες, ταξινόμηση με σωρό.

Αλγόριθμοι και Πολυπλοκότητα

Μάθετε περισσότερα για το αντικείμενο

Το εισαγωγικό αυτό μάθημα αλγορίθμων έχει σαν βασικό στόχο να αποκτήσουν οι ενδιαφερόμενοι που το παρακολουθούν αλγοριθμική σκέψη (algorithmic thinking). Για την επίτευξη αυτού του στόχου το μάθημα επικεντρώνεται στην αφομοίωση του τι σημαίνει καλός αλγόριθμος, πως μετράμε πόσο γρήγορος είναι ένας αλγόριθμος και ποια είναι η βασική μαθηματική ιδέα στην οποία βασίζεται ένας αλγόριθμος.

- 2 ώρες / βδομάδα

- 50€ / 6 μήνες

- Περιεχόμενο μαθήματος:

  • Εισαγωγή, Ασυμπτωτικοί Συμβολισμοί, Σωροί, Παραδείγματα υπολογισμού πολυπλοκότητας αλγορίθμων
  • Διαίρει-και-Βασίλευε
  • Άπληστοι Αλγόριθμοι
  • Δυναμικός Προγραμματισμός
  • Αλγόριθμοι Γράφων
  • ΝΡ-πληρότητα

Εισαγωγή στον Προγραμματισμό στον Ιστό

Μάθετε περισσότερα για το αντικείμενο

Το αντικείμενο του μαθήματος αφορά τον προγραμματισμό της ιστοσελίδας με τις κατάλληλες γλώσσες προγραμματισμού και την κατανόηση των βασικών εννοιών που ισχύουν στον Παγκόσμιο Ιστό.

- 2 ώρες / βδομάδα

- 50€ / 6 μήνες

- Περιεχόμενο μαθήματος:

  • Γνωριμία με έννοιες και πρότυπα του Παγκοσμίου Ιστού
  • Κατανόηση της αρχιτεκτονικής του Παγκοσμίου Ιστού και του ρόλου του πρωτοκόλλου HTTP
  • Εκμάθηση της χρήσης των HTML5 και CSS για την παραγωγή περιεχομένου του Παγκοσμίου Ιστού
  • Εισαγωγή στον προγραμματισμό με χρήση JavaScript
  • Κατανόηση των ιδιαιτεροτήτων της JavaScript σε σχέση με αντικειμενοστρεφείς γλώσσες
  • Εξοικείωση με βασικές έννοιες και προγραμματιστικές διεπαφές
  • Εκμάθηση της χρήσης JavaScript για υλοποίηση εφαρμογών στην πλατφόρμα Web Browser
  • Εισαγωγή στην υλοποίηση διακομιστών ιστού στην πλατφόρμα Node.js με χρήση JavaScript
  • Κατανόηση της χρήσης του HTTP και βασικών μορφοτύπων περιεχομένων για τη σχεδίαση υπηρεσιών ιστού
